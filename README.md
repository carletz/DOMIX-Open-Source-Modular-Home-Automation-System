# DOMIX-Open-Source-Modular-Home-Automation-System
Open-source modular DIN-rail home automation. ESP32-S3 core with Ethernet, Zigbee, RS485. 9 I/O modules via I²C: relays, contacts, energy meter, sprinklers. ESPHome-based, integrates Home Assistant. Low-voltage, cabinet-oriented. Built for learning &amp; customization.
